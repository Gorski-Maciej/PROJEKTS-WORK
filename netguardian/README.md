# NetGuardian 2.0 – Autonomiczny System Monitoringu i Reakcji Sieciowej

![Security](https://img.shields.io/badge/Security-active-red)

NetGuardian to zaawansowana platforma SIEM/SOAR, która monitoruje ruch sieciowy w czasie rzeczywistym, wykrywa anomalie i automatycznie reaguje, blokując źródło ataku, wdrażając honeypoty i generując raporty.

## 🚀 Kluczowe cechy
- **Detekcja anomalii ML**
- **Zaawansowane detektory** – beaconing, DNS exfiltration, data exfiltration, korelacja zdarzeń
- **Automatyczna reakcja** – blokada IP, playbooki SOAR, powiadomienia
- **Threat Intelligence** – integracja z AbuseIPDB i MISP
- **Dashboard czasu rzeczywistego**

## ⚡ Szybki start
```bash
cd netguardian
cp .env.example .env
bash scripts/setup.sh
```
