# NetAegis – Platforma Automatyzacji Bezpieczeństwa Sieci (SOAR)

![SOAR](https://img.shields.io/badge/SOAR-platform-orange)

NetAegis to rozproszona platforma typu SOAR składająca się z wielu agentów zbierających dane z sieci i systemów, centralnego silnika decyzyjnego (MCP) oraz interfejsu dowodzenia.

## 🚀 Kluczowe cechy
- **Architektura agentowa**
- **Master Control Program (MCP)**
- **Operational MCP**
- **Playbooki SOAR**
- **Dashboard React**

## ⚡ Szybki start
```bash
cd netaegis
cp .env.example .env
bash scripts/setup.sh
```
