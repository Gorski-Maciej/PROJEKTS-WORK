Wyobraź sobie firmę z wieloma oddziałami. Każdy oddział ma lokalną infrastrukturę (serwery, sieć), która musi działać autonomicznie, ale centralny zespół IT potrzebuje jednolitego widoku i możliwości zdalnego zarządzania. W tym momencie wkracza architektura MCP.

Główny Serwer MCP (Centralny "Mózg"): Jest to centralny punkt dowodzenia. Przetrzymuje globalną konfigurację, polityki bezpieczeństwa, modele predykcyjne i bazę wiedzy o całej infrastrukturze. Odpowiada za koordynację działań, agregację danych i podejmowanie strategicznych decyzji.

🚀 Genialna Funkcjonalność: "Tryb Awaryjny" (Autonomous Mode)
To jest funkcja, która robi wrażenie na rozmowie kwalifikacyjnej.
Scenariusz: Łącze internetowe w oddziale firmy zostaje przerwane.

Operacyjny Serwer MCP na miejscu wykrywa utratę połączenia z Głównym Serwerem MCP.

Automatycznie przechodzi w tryb autonomiczny, logując zdarzenie.

Nadal normalnie zbiera dane od lokalnych agentów i, co najważniejsze, wykonuje zapisane lokalnie polityki bezpieczeństwa. Jeśli w tym czasie nastąpi atak brute-force na lokalny serwer, Operacyjny MCP samodzielnie zablokuje atakującego na lokalnym firewallu.

Gdy połączenie zostanie przywrócone, Operacyjny MCP synchronizuje wszystkie zebrane dane i logi zdarzeń z Głównym Serwerem MCP, raportując "co się działo pod jego nieobecność".

4.3. Agenci (Oczy i Ręce Systemu)
Każdy agent to niezależna, lekka usługa systemd, napisana w Pythonie.

NetPulse Agent:

Wykonuje cykliczne kontrole: ping, traceroute, skanowanie portów (nmap), sprawdza zużycie CPU/RAM/dysku.

Wyniki wysyła do lokalnego Operacyjnego MCP.

Lekka implementacja: wykorzystuje asyncio do nieblokującego I/O.

SecLog Agent:

Monitoruje w czasie rzeczywistym pliki logów (syslog, auth.log, nginx) za pomocą biblioteki watchfiles.

Parsuje zdarzenia i klasyfikuje je (np. "failed_login", "new_connection").

Wysyła zdarzenia do Operacyjnego MCP.

NetConfig Agent:

Zbiera konfiguracje urządzeń sieciowych przez SSH (paramiko).

Wykorzystuje GitPython do wersjonowania zmian w lokalnym repozytorium Git.

Obsługuje szablony Jinja2 do generowania i wypychania nowych konfiguracji.

🚀 5. Kluczowe mechanizmy i funkcjonalności
5.1. Autonomiczna Pętla Reakcji (Self-Healing Loop)
Pokazuje pełną automatyzację procesu.

Monitorowanie: NetPulse Agent wykrywa, że serwer nie odpowiada na ping.

Analiza i Decyzja: Lokalny Operacyjny MCP otrzymuje alert i uruchamia Rule Engine. Reguła "Host Unreachable" ma zdefiniowaną akcję.

Wykonanie: Operacyjny MCP wysyła do NetConfig Agenta zadanie: "połącz się przez SSH z serwerem i wykonaj polecenie restartu usługi".

Weryfikacja i Raport: NetPulse Agent potwierdza, że ping powrócił. Incydent jest automatycznie zamykany, a do administratora wysyłane jest podsumowanie na Slack.

5.2. Tryb Awaryjny (Offline Operation)
Dowodzi odporności systemu.

Operacyjny MCP stale monitoruje połączenie z Głównym MCP (heartbeat).

Gdy połączenie zostanie zerwane, przechodzi w tryb autonomiczny, loguje zdarzenie i kontynuuje wykonywanie lokalnych polityk.

Wszystkie nowe dane są kolekcjonowane w lokalnej bazie SQLite.

Po przywróceniu łączności, Operacyjny MCP synchronizuje wszystkie zaległe dane i zdarzenia z Głównym MCP, używając mechanizmu "catch-up" w NATS JetStream.

5.3. Silnik Reguł (Rule Engine)
Reguły są definiowane w plikach YAML, co pozwala na ich modyfikację bez restartu aplikacji.

Przykład reguły:

yaml
name: "Brute Force Detection"
conditions:
  - field: "event_type"
    operator: "equals"
    value: "failed_login"
  - field: "source_ip"
    operator: "same_in_timeframe"
    count: 5
    timeframe_seconds: 60
actions:
  - type: "ssh_command"
    target: "firewall_main"
    command: "block_ip {{ source_ip }}"
  - type: "notification"
    channel: "slack"
    message: "Zablokowano IP {{ source_ip }} za atak brute-force."
5.4. Zarządzanie Konfiguracją (Git-backed)
Każda zmiana konfiguracji urządzenia jest automatycznie commitowana do lokalnego repozytorium Git.

Panel umożliwia przeglądanie historii, diff-ów i przywracanie poprzednich wersji jednym kliknięciem.

System szablonów Jinja2 pozwala na wdrożenie tej samej konfiguracji na wiele urządzeń, dostosowując tylko zmienne (np. adres IP).

import bcrypt
from sqlalchemy import Column, String, Boolean
from shared.models import Base

class User(Base):
    __tablename__ = "users"
    id = Column(String, primary_key=True, default=lambda: str(__import__("uuid").uuid4()))
    username = Column(String, unique=True, nullable=False)
    hashed_password = Column(String, nullable=False)
    role = Column(String, default="admin")
    is_active = Column(Boolean, default=True)

    @staticmethod
    def hash_password(password: str) -> str:
        return bcrypt.hashpw(password.encode(), bcrypt.gensalt()).decode()

    def verify_password(self, password: str) -> bool:
        return bcrypt.checkpw(password.encode(), self.hashed_password.encode())

from typing import Optional

import asyncio

class RuleEngine:
    """Silnik oceniający warunki i zwracający akcje do wykonania."""

    def __init__(self):
        self._cache: list[dict] = []

    def load_rules(self, rules: list[dict]):
        """Ładuje reguły do pamięci podręcznej (wywoływane po zmianach w bazie)."""
        self._cache = sorted(rules, key=lambda r: r.get("priority", 1))

    def evaluate(self, event: dict) -> Optional[list[dict]]:
        """Sprawdza zdarzenie wobec reguł. Zwraca listę akcji lub None."""
        for rule in self._cache:
            if self._match_condition(event, rule["condition"]):
                return rule.get("actions", [rule["action"]])
        return None

    def _match_condition(self, event: dict, condition: dict) -> bool:
        field = condition.get("field")
        operator = condition.get("operator", "equals")
        value = condition.get("value")

        event_value = event.get(field)
        if event_value is None:
            return False

        if operator == "equals":
            return str(event_value) == str(value)
        elif operator == "contains":
            return str(value) in str(event_value)
        return False

import duckdb
from datetime import datetime, timedelta
from prophet import Prophet
import pandas as pd

class Predictor:
    """Prognozuje trendy na podstawie danych historycznych."""

    def __init__(self, db_path: str = "analytics.duckdb"):
        self.db_path = db_path

    def train_and_forecast(self, metric_name: str = "cpu_usage", days_history: int = 30, days_ahead: int = 7) -> dict:
        conn = duckdb.connect(self.db_path)
        try:
            df = conn.execute(f"""
                SELECT timestamp::DATE AS ds, AVG(value) AS y
                FROM metrics
                WHERE name = '{metric_name}'
                  AND timestamp >= CURRENT_DATE - INTERVAL {days_history} DAY
                GROUP BY 1
                ORDER BY 1
            """).df()
        finally:
            conn.close()

        if df.empty:
            return {"error": "Brak danych do prognozy"}

        df.columns = ["ds", "y"]
        df["ds"] = pd.to_datetime(df["ds"])

        model = Prophet(daily_seasonality=True)
        model.fit(df)

        future = model.make_future_dataframe(periods=days_ahead)
        forecast = model.predict(future)

        result = forecast[["ds", "yhat", "yhat_lower", "yhat_upper"]].tail(days_ahead)
        return result.to_dict(orient="records")

import asyncio
import json
import nats
from nats.js.api import ConsumerConfig, AckPolicy
import duckdb
from datetime import datetime

class Aggregator:
    """Nasłuchuje na NATS i zapisuje metryki do DuckDB."""

    def __init__(self, nats_url: str = "nats://localhost:4222", duckdb_path: str = "analytics.duckdb"):
        self.nats_url = nats_url
        self.duckdb_path = duckdb_path
        self._running = False

    async def start(self):
        self._running = True
        self.nc = await nats.connect(self.nats_url)
        self.js = self.nc.jetstream()

        # Subskrypcja na metryki
        await self.js.subscribe("agent.metrics", cb=self._on_metric, durable="aggregator",
                                config=ConsumerConfig(ack_policy=AckPolicy.EXPLICIT))

        # Subskrypcja na logi
        await self.js.subscribe("agent.logs", cb=self._on_log, durable="aggregator-logs",
                                config=ConsumerConfig(ack_policy=AckPolicy.EXPLICIT))

        print("Aggregator: Nasłuchiwanie rozpoczęte...")
        while self._running:
            await asyncio.sleep(1)

    async def stop(self):
        self._running = False
        await self.nc.close()

    async def _on_metric(self, msg):
        data = json.loads(msg.data.decode())
        conn = duckdb.connect(self.duckdb_path)
        conn.execute("""
            INSERT INTO metrics (timestamp, agent_id, name, value)
            VALUES (?, ?, ?, ?)
        """, [datetime.now(), data.get("agent_id"), data.get("name"), data.get("value")])
        conn.close()
        await msg.ack()

    async def _on_log(self, msg):
        data = json.loads(msg.data.decode())
        conn = duckdb.connect(self.duckdb_path)
        conn.execute("""
            INSERT INTO log_events (timestamp, agent_id, event_type, details)
            VALUES (?, ?, ?, ?)
        """, [datetime.now(), data.get("agent_id"), data.get("type"), json.dumps(data)])
        conn.close()
        await msg.ack()

import asyncio
import logging
import httpx
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession
from models import LocalEvent

logger = logging.getLogger(__name__)

class AutonomousManager:
    """Monitoruje łączność z Głównym MCP i zarządza trybem autonomicznym."""

    def __init__(self, main_mcp_url: str, session_factory):
        self.main_mcp_url = main_mcp_url
        self.session_factory = session_factory
        self.is_autonomous = False
        self._task = None

    async def start(self):
        self._task = asyncio.create_task(self._monitor_loop())

    async def stop(self):
        if self._task:
            self._task.cancel()

    async def _monitor_loop(self):
        """Co 10 sekund próbuje połączyć się z Głównym MCP."""
        while True:
            try:
                async with httpx.AsyncClient() as client:
                    resp = await client.get(f"{self.main_mcp_url}/api/op/heartbeat", timeout=5.0)
                    if resp.status_code == 200:
                        if self.is_autonomous:
                            logger.info("Połączenie przywrócone. Synchronizacja danych...")
                            await self._sync_data()
                        self.is_autonomous = False
                    else:
                        self.is_autonomous = True
            except Exception:
                self.is_autonomous = True
                logger.warning("Brak połączenia z Głównym MCP. Przejście w tryb autonomiczny.")

            await asyncio.sleep(10)

    async def _sync_data(self):
        """Wysyła wszystkie niezsynchronizowane zdarzenia do Głównego MCP."""
        async with self.session_factory() as session:
            result = await session.execute(select(LocalEvent).where(LocalEvent.synced == False))
            events = result.scalars().all()

            if events:
                async with httpx.AsyncClient() as client:
                    payload = [{"type": e.type, "payload": e.payload} for e in events]
                    await client.post(f"{self.main_mcp_url}/api/op/sync", json={"events": payload})

                for event in events:
                    event.synced = True
                await session.commit()

                logger.info(f"Zsynchronizowano {len(events)} zdarzeń.")

import logging
import paramiko
from models import ActionQueue

logger = logging.getLogger(__name__)

class ActionExecutor:
    """Wykonuje zadania z kolejki (np. blokowanie IP przez SSH)."""

    async def execute(self, action: ActionQueue) -> bool:
        """Wykonuje pojedynczą akcję i zwraca True przy sukcesie."""
        logger.info(f"Wykonywanie akcji {action.id}: {action.action_type} na {action.target}")

        if action.action_type == "ssh_command":
            return await self._exec_ssh(action.target, action.params.get("command", ""))

        elif action.action_type == "block_ip":
            ip = action.params.get("ip")
            return await self._exec_ssh(action.target, f"iptables -A INPUT -s {ip} -j DROP")

        logger.error(f"Nieznany typ akcji: {action.action_type}")
        return False

    async def _exec_ssh(self, host: str, command: str) -> bool:
        """Wykonuje polecenie przez SSH."""
        try:
            ssh = paramiko.SSHClient()
            ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
            ssh.connect(host, username="admin", password="password", timeout=10)
            stdin, stdout, stderr = ssh.exec_command(command)
            output = stdout.read().decode()
            ssh.close()
            logger.info(f"SSH({host}): {command} -> {output}")
            return True
        except Exception as e:
            logger.error(f"SSH({host}) błąd: {e}")
            return False

def ping_host(ip: str, count: int = 1) -> float | None:
    """Pinguje host i zwraca RTT w ms lub None."""
    try:
        result = subprocess.run(["ping", "-c", str(count), ip],
                                capture_output=True, text=True, timeout=5)
        if result.returncode == 0:
            match = re.search(r"time=(\d+\.?\d*) ms", result.stdout)
            if match:
                return float(match.group(1))
    except Exception:
        pass
    return None

def check_port(ip: str, port: int, timeout: float = 2.0) -> bool:
    """Sprawdza, czy port jest otwarty."""

            return True
    except Exception:
        return False

async def send_log_event(event_type: str, details: dict):
    """Wysyła zdarzenie do Operacyjnego MCP."""
    async with httpx.AsyncClient() as client:
        await client.post(
            f"{OPERATIONAL_MCP_URL}/api/agents/logs",

    for key, parser in PARSERS.items():
        if key in path:
            return parser
    return None

async def main():
    print(f"[{AGENT_ID}] Uruchomiony. Monitorowanie: {WATCH_PATHS}")

    async for changes in awatch(*WATCH_PATHS):
        for change_type, path in changes:
            parser = get_parser_for_path(path)
            if parser is None:
                continue

            try:
                with open(path, "r") as f:
                    # W uproszczeniu: czytamy ostatnią linię
                    lines = f.readlines()
                    if lines:
                        last_line = lines[-1].strip()
                        event = parser(last_line)
                        if event:
                            await send_log_event(event["type"], event["details"])
                            print(f"[{AGENT_ID}] Zdarzenie: {event['type']}")
            except Exception as e:
                print(f"[{AGENT_ID}] Błąd odczytu {path}: {e}")

async def collect_config(device_ip: str, username: str = "admin", password: str = "admin") -> str:
    """Pobiera konfigurację z urządzenia sieciowego przez SSH."""
    ssh = paramiko.SSHClient()
    ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
    ssh.connect(device_ip, username=username, password=password, timeout=10)
    stdin, stdout, stderr = ssh.exec_command("show running-config")
    config = stdout.read().decode()
    ssh.close()
    return config

async def push_config(device_ip: str, config: str, username: str = "admin", password: str = "admin"):
    """Wysyła konfigurację na urządzenie."""
    ssh = paramiko.SSHClient()
    ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
    ssh.connect(device_ip, username=username, password=password, timeout=10)
    ssh.exec_command("configure terminal")
    for line in config.split("\n"):
        ssh.exec_command(line)
    ssh.exec_command("end")
    ssh.close()

def render_template(template_name: str, context: dict) -> str:
    """Renderuje szablon Jinja2 z kontekstem."""
    env = Environment(loader=FileSystemLoader(str(TEMPLATES_DIR)))
    template = env.get_template(template_name)
    return template.render(context)

frontend/src/hooks/useWebSocket.js
javascript
import { useEffect, useRef } from 'react';

export default function useWebSocket(url, onMessage) {
  const wsRef = useRef(null);

  useEffect(() => {
    const protocol = window.location.protocol === 'https:' ? 'wss:' : 'ws:';
    const wsUrl = `${protocol}//${window.location.host}${url}`;

    wsRef.current = new WebSocket(wsUrl);
    wsRef.current.onmessage = (event) => {
      try {
        const data = JSON.parse(event.data);
        onMessage(data);
      } catch (e) {
        console.error('WebSocket parse error:', e);
      }
    };

    wsRef.current.onerror = (e) => console.error('WebSocket error:', e);

    return () => {
      wsRef.current?.close();
    };
  }, [url]);

  return wsRef;
}
